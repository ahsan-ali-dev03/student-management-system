# 🎓 Student Management System

A complete Java + MySQL based desktop application for managing student records efficiently.

---

# 🚀 Features

✅ Add Student  
✅ Update Student Details  
✅ Delete Student  
✅ Search Student  
✅ Login System  
✅ Database Connectivity  
✅ GUI Interface using Java Swing  
✅ MySQL Integration  

---

# 🛠️ Technologies Used

- Java ☕
- MySQL 🗄️
- JDBC 🔌
- Java Swing 🎨
